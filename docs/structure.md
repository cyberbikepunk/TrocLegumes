# TrocLégumes — Project Structure

## Overview

Django separates the project **configuration** (in `config/`) from the **business applications** (in `apps/`).
Each app is a self-contained Python module with its own models, views, URLs, and templates.

---

## Key Django Concepts

| Concept | File(s) | Role |
|---|---|---|
| **Model** | `apps/*/models.py` | Defines database tables. Django generates the SQL. |
| **Migration** | `apps/*/migrations/` | Schema change history. `make migrate` applies them. |
| **View** | `apps/*/views.py` | Receives an HTTP request, returns a response (HTML, JSON…) |
| **URL** | `config/urls.py` + `apps/*/urls.py` | Maps a URL pattern to a view |
| **Template** | `templates/**/*.html` | HTML with Django tags `{{ variable }}` and `{% tag %}` |
| **Form** | `apps/*/forms.py` | Validates user-submitted data |
| **Admin** | `apps/*/admin.py` | Registers models into the `/admin` interface |

---

## HTTP Request Flow

```
Browser → URL → config/urls.py → apps/xxx/urls.py → View → Template → HTML
                                                       ↕
                                                    Model (DB)
```

---

## Useful Commands

```bash
make up              # Start Docker containers (web + db + redis)
make down            # Stop everything

make shell           # Open a shell inside the web container
make migrate         # Apply pending migrations
make makemigrations  # Generate migrations after editing a models.py
make superuser       # Create an admin account
make test            # Run pytest
make lint            # Run ruff (linter)
```

## Environment Variables (`.env` file)

Never committed to git. See `.env.example` for the template.

| Variable | Dev value | Role |
|---|---|---|
| `DEBUG` | `True` | Django debug mode |
| `SECRET_KEY` | (random key) | Signs sessions and tokens |
| `DATABASE_URL` | `postgres://…@db:5432/troclegumes` | PostgreSQL connection |
| `REDIS_URL` | `redis://redis:6379/0` | Celery broker |
| `ALLOWED_HOSTS` | `localhost,127.0.0.1` | Allowed hostnames |
